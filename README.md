#  Employee Salary Prediction using Machine Learning

## 📌 Project Overview

This project predicts employee salaries based on job-related information such as experience, education level, job title, industry, company size, location, remote work status, certifications, and skills.

The model was trained using the Random Forest Regressor algorithm and deployed as a Streamlit web application.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Streamlit
- Joblib

-----

## 📊 Dataset Features

- Job Title
- Experience Years
- Education Level
- Skills Count
- Industry
- Company Size
- Location
- Remote Work
- Certifications

Target Variable:

- Salary

-----

## 🤖 Machine Learning Model

Random Forest Regressor

---

## 📈 Model Performance

| Metric | Score |
|---------|--------|
| MAE | 5693.02 |
| RMSE | 7375.76 |
| R² Score | 96.09% |

The model achieved excellent prediction performance.

## ▶️ Run Locally

Clone the repository

```bash
git clone https://github.com/YourUsername/employee-salary-prediction.git
```

Move to the project folder

```bash
cd employee-salary-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit app

```bash
streamlit run app.py
```
